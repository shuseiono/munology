## 6. プライバシー・計測

| 項目 | 現状・注意 |
|------|------------|
| GTM | コンテナ ID: `GTM-TLDDNS2X` |
| GA4 | 測定 ID: `G-1K5P3M2GMT` |
| 実装方針 | **GTM を正とし、GTM 内で GA4 を配信する前提**。二重計測を避けるため、単体 gtag 埋め込みは撤去済みの想定。**GTM に GA4 が未設定の場合は GTM 側の設定を先に直すこと。** |
| Cookie 同意 | プライバシーポリシー・地域規制に応じてバナー要否を別途判断 |

## 7. コンテンツ運用

| 項目 | 合意の方向 |
|------|------------|
| 記事追加 | `src/content/articles/*.md`、フロントマターは `src/content/config.ts` の Zod スキーマに準拠 |
| 下書き | `draft: true` で本番ビルドから除外 |
| 画像 | 原則 `public/images/`。OG 用 `image` フィールドの URL は絶対 URL 推奨 |
