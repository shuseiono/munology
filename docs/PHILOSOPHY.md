## 1. 何のために在るか

模擬国連は、国際政治・国際法・議論・交渉の一端を、限られた時間のなかで **実践** することにより、体験する場です。議題は整理され、役割は事前にある程度指定され、そのなかで思考し、実践することを求められます。

その一方で、どんなに長期間かけて準備をした会議であっても、**いつか終わる**し、会議を通して学んだことも、その全容は会議の中に到底収まりきらない。Ex.条文の解釈、歴史的経緯、学説の争い、現実の外交戦略——これらを「勝つための材料」に還元してしまうと、せっかく幅広い学問の一端に触れられる、という模擬国連のメリット=学びの厚みが消失する。

MUNOLOGYは、**会議の外側・あと**まで続く学びを、高校以上の探究者のために整えた **編集型の読み物** です。法学・政治学の枠組みを明示し、**根拠と論点が追える言葉**で、国際法・国際政治を深く理解する手助けをします。

## 2. 「DRの向こう側」と英語副題 *Beyond the Gavel*

### 2.1 日本語で言っていること

- **会議（＝会期・委員会のセッション）**  
  会議中の学びは貴重ですが、時間・手続・役割の制約の中にある。

- **向こう側**  
  （大使としての人格を取り外した状態で）会議が一旦終わったあと、あるいは会議用の要約（DR 等）の **更に外側** で、**なぜそう解釈されるのか、どの前提が立っているのか** を問い直し、地続きの教養にすることを指します。  
  目標は「会議用テクニックの量産」ではなく、**会議体験を、より持続的な理解へ接続する** ことです。

### 2.2 英語表記

**公式副題**は ***Beyond the Gavel*** とする。ガベルは**会期の閉会・終了**の象徴で、MUN 文脈で身体化されやすく、日本語の「会議のあと」に手触りで近い。*conference* ほど他義へのぶれが少なく、副題帯向きの長さで **略称に頼りにくい**（公式では常にスペルアウトする）。表記は README・フッター・ヘッダー等で揃え、[Design.md](../Design.md) にも従う。

## 3. どの場所を占めるか（位置づけ）

| MUNOLOGY が主に担うこと | 意図的に主役にしないこと（例） |
|-------------------------|----------------------------------|
| 条約・判例・概念の、根拠つきの解説 | 会場の噂話・個人攻撃的な中傷 |
| 探究に耐える文章とカテゴリ（最新動向・頻出トピック・コア文書・学問的探究 等） | 浅い釣り文句だけのまとめ |
| 会議直後の学びの「接続端子」 | 大学以降の高等教育そのものの代替を自称すること |

「模擬国連向けの情報」は広く取れます。MUNOLOGYはそのうち、**学術的・エディトリアルな厳しさ**と **web での到達** を両立しようとする、特定の帯域を選びます。選び方の詳細は [Design.md](../Design.md) のトーンと禁止事項に表れています。

## 4. 倫理と公開性（価値の根）

1. **教育資源へのアクセスに差がある現実** を前提に、質の高い解説を **可能な形でオープンに届ける** ことに意味があります。大学の講義に物理的に行けない読者も、**同等の深さの議論に足を踏み入れられる材料** を目指します。  
2. **次の世代** — 国際社会で責任を持って働く人材の育成に、**コンテンツの面**から補助的に貢献したい、という願いを抱きます。  
3. 上記は宣伝文句ではなく、**採択・表現・掲載方針の最後に戻るべき物差し** です。
